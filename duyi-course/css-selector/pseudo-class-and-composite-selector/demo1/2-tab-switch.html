示例2: 选项卡切换 (input[type="radio"]:checked + .tab-label + .tab-content)

这个示例使用了:checked伪类和相邻兄弟选择器+来创建无JavaScript的标签切换效果。 当单选按钮被选中时，相邻的标签和内容会改变样式。

input[type="radio"]:checked + .tab-label { background-color: #3498db; color: white; } input[type="radio"]:checked + .tab-label + .tab-content { display: block; }
