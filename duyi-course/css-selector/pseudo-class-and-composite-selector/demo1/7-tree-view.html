示例7: 树形结构菜单 (input[type="checkbox"]:checked ~ .tree-children)

这个示例使用了:checked伪类和一般兄弟选择器~来创建可折叠的树形结构菜单。 当复选框被选中时，子菜单会展开显示。

.tree-view input[type="checkbox"]:checked ~ .tree-children { display: block; } .tree-view input[type="checkbox"]:checked ~ .tree-item::before { transform: rotate(90deg); }
