示例5: 手风琴/折叠面板 (input[type="checkbox"]:checked ~ .accordion-content)

这个示例使用了:checked伪类和一般兄弟选择器~来创建无JavaScript的手风琴/折叠面板效果。 当复选框被选中时，相关内容会展开显示。

.accordion-item input[type="checkbox"]:checked ~ .accordion-content { max-height: 500px; } .accordion-item input[type="checkbox"]:checked ~ .accordion-header::after { content: "-"; }

这是第一部分的内容。点击标题可以展开或折叠内容。这个效果完全使用CSS实现，不需要JavaScript。

这是第二部分的内容。每个部分可以单独展开或折叠。

这是第三部分的内容。手风琴/折叠面板对于需要节省空间但又希望提供详细信息的场景非常有用。
