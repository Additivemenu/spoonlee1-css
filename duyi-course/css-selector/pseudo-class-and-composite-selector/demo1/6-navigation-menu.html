示例6: 导航菜单高亮当前页面 (.nav li.active a, .nav li a:hover)

这个示例展示了如何使用class属性结合:hover伪类来高亮显示当前页面和悬停状态的导航菜单项。

.nav li a:hover { background-color: #34495e; } .nav li.active a { background-color: #3498db; font-weight: bold; } .nav li.active a:hover { background-color: #2980b9; }
