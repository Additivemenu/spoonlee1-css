伪类与组合选择器实例演示

示例4: 表单验证和状态 (.form-control:invalid + .form-message)

这个示例使用了:valid和:invalid伪类，结合相邻兄弟选择器+来显示表单验证消息。 当输入无效时，会显示错误消息。

.form-control:focus { outline: none; border-color: #3498db; box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3); } .form-control:valid { border-color: #2ecc71; } .form-control:invalid { border-color: #e74c3c; } .form-control:invalid + .form-message { display: block; color: #e74c3c; }
