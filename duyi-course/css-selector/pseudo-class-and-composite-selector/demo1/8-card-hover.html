伪类与组合选择器实例演示

示例8: 卡片悬停效果 (.card:hover, .card:hover .card-title)

这个示例展示了如何使用:hover伪类和后代选择器来创建卡片的悬停效果。 当鼠标悬停在卡片上时，卡片会上移并且标题颜色会改变。

.card:hover { transform: translateY(-10px); box-shadow: 0 12px 20px rgba(0,0,0,0.2); } .card:hover .card-title { color: #3498db; }
