CSS 伪元素选择器演示

::before 伪元素

伪元素 ::before 用于在元素内容之前插入内容。通常使用 content 属性定义要插入的内容。

.before-demo::before {
  content: "⚠️";
  position: absolute;
  left: -15px;
  top: 50%;
  transform: translateY(-50%);
  font-size: 24px;
}
这是一个使用 ::before 添加图标的示例，注意左侧的警告图标是通过 ::before 添加的。

::after 伪元素

伪元素 ::after 用于在元素内容之后插入内容。与 ::before 类似，也使用 content 属性。

.after-demo::after {
  content: "→";
  position: absolute;
  right: 0;
  top: 50%;
  transform: translateY(-50%);
  font-size: 20px;
}

点击查看更多 (注意右侧的箭头是通过 ::after 添加的)

::first-line 伪元素

伪元素 ::first-line 用于选择元素的第一行文本，即使文本换行，也只会影响第一行。

.first-line-demo::first-line {
  font-weight: bold;
  color: #e74c3c;
  font-size: 1.2em;
}

这是一段示例文本，可以看到第一行的样式与其他行不同。这段文字比较长，会自动换行，但只有第一行会应用特殊样式。不论屏幕大小如何变化，或者窗口大小如何调整，始终只有第一行会应用这个特殊样式。

::first-letter 伪元素

伪元素 ::first-letter 用于选择元素文本的第一个字母。

.first-letter-demo::first-letter {
  font-size: 2.5em;
  font-weight: bold;
  color: #9b59b6;
  float: left;
  line-height: 0.8;
  margin-right: 5px;
}

这个段落的第一个字母有特殊样式，就像书籍中常见的首字下沉效果一样。这种效果在杂志和网站中经常用于强调段落的开始。

::selection 伪元素

伪元素 ::selection 用于定义用户选中文本时的样式。

.selection-demo::selection {
  background-color: #f1c40f;
  color: #2c3e50;
}

试着用鼠标选中这段文字，你会看到自定义的选中效果 - 黄色背景和深色文字，而不是浏览器默认的蓝底白字。

::placeholder 伪元素

伪元素 ::placeholder 用于设置表单元素的占位符文本样式。

.placeholder-demo::placeholder {
  color: #7f8c8d;
  font-style: italic;
}

组合应用示例

1. 装饰性角框

.decorative-box::before,
.decorative-box::after {
  content: "";
  position: absolute;
  width: 20px;
  height: 20px;
}

.decorative-box::before {
  top: -10px;
  left: -10px;
  border-top: 3px solid #e74c3c;
  border-left: 3px solid #e74c3c;
}

.decorative-box::after {
  bottom: -10px;
  right: -10px;
  border-bottom: 3px solid #e74c3c;
  border-right: 3px solid #e74c3c;
}
这个盒子使用了 ::before 和 ::after 伪元素创建左上角和右下角的装饰性边框，无需添加额外的HTML元素。

2. 引用样式

.quote::before {
  content: """;
  font-size: 2em;
  color: #3498db;
  position: absolute;
  left: -5px;
  top: -10px;
}

.quote::after {
  content: """;
  font-size: 2em;
  color: #3498db;
  position: absolute;
  right: -5px;
  bottom: -20px;
}
伪元素是CSS中非常强大的功能，它可以让我们在不增加额外HTML元素的情况下，添加装饰性内容或实现特殊效果。

3. 动态链接效果

.dynamic-link::after {
  content: "";
  position: absolute;
  bottom: 0;
  left: 0;
  width: 0;
  height: 2px;
  background-color: #2980b9;
  transition: width 0.3s ease;
}

.dynamic-link:hover::after {
  width: 100%;
}

尝试将鼠标悬停在这个动态链接上，观察下划线的动画效果。
